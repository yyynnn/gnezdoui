# Toggle

## Basic Usage

<Playground>
  {() => {
    const Component = () => {
      const [state, setState] = useState(true)
      return <Toggle rightText={'Text'} hint={'Hint'} checked={state} onChange={() => setState(!state)} />
    }
    return <Component />
  }}
</Playground>
